## **How to use**
- open your main.js file
- type it as module file
```javascript
import slideInit from './slider.js' ;
const slider = document.querySelector(".slider") ;
slideInit(slider) ;
```

**Link CSS**
```HTML
<link rel="stylesheet" href="main.css" />
```
